# 中西藥併用風險評估功能 - 使用說明

## 🎯 功能概述

本功能允許病患回報目前使用的中藥材/方劑/保健品，系統會自動從 FHIR 病歷中取得西藥處方（優先使用 MedicationDispense），並根據預設的交互作用規則表進行風險評估，提供風險分級與就醫建議。

## 📱 使用流程

### 1. 登入系統
- 使用 SMART on FHIR 認證登入
- 系統會自動載入您的病患資料

### 2. 選擇中藥材
- **搜尋**：在搜尋框輸入中藥材名稱或別名（支援模糊搜尋）
- **分類瀏覽**：點選分類標籤（活血、補氣、清熱等）
- **勾選**：點擊核取方塊選擇正在使用的中藥材
- **補充資訊**（勾選後顯示）：
  - 開始日期（必填建議）
  - 使用頻率：每日/每週/不定期
  - 來源：藥行抓藥/自煎/科學中藥/保健品/其他

### 3. 開始風險評估
- 點擊「開始評估風險」按鈕
- 系統會執行以下步驟：
  1. 驗證您回報的中藥材資料
  2. 從病歷取得您的西藥處方
  3. 比對交互作用規則
  4. 產生風險評估報告

### 4. 查看結果
- **整體風險等級**：低/中/高
- **檢出的風險項目**：
  - 中藥 + 西藥組合
  - 風險類型（如：出血風險、低血糖風險）
  - 風險嚴重度
  - 提示訊息與建議行動
  - 警訊症狀（高風險項目）
- **用藥摘要**：
  - 您回報的中藥清單
  - 從病歷取得的西藥清單

### 5. 後續動作
- **重新評估**：修改中藥選擇後重新評估
- **匯出報告**：下載或列印評估結果（TODO）
- **就醫諮詢**：依建議諮詢醫師或藥師

## ⚠️ 重要提示

### 本系統的定位
- ✅ **是**：用藥安全風險提示與照護協作工具
- ❌ **不是**：醫療診斷、治療建議或處方系統

### 免責聲明
1. 此為「風險提示」，非醫療診斷或治療建議
2. 中藥使用資訊為病人自行回報，可能不完整
3. 如有不適或疑慮，請洽詢醫師或藥師
4. 不建議自行停藥、換藥或調整劑量

### 資料安全
- 您的資料透過 SMART on FHIR 安全傳輸
- 評估結果僅供您個人參考
- 系統不會將您的資料分享給第三方

## 📊 風險分級說明

### 🔴 高風險
- **定義**：已知高風險組合，可能造成嚴重健康影響
- **範例**：活血類中藥 + 抗凝血藥（出血風險顯著增加）
- **建議**：儘速（24-48小時內）諮詢醫師或藥師
- **注意**：會顯示警訊症狀清單

### 🟡 中風險
- **定義**：有合理機轉/常見併用風險
- **範例**：人參 + 降血糖藥（可能增強降血糖效果）
- **建議**：留意症狀並諮詢醫療專業人員
- **注意**：需定期監測相關指標（如血糖、血壓）

### 🟢 低風險
- **定義**：可能性低或證據弱
- **範例**：無匹配的交互作用規則
- **建議**：以衛教提醒為主
- **注意**：仍建議在就醫時告知所有用藥

## 🗂️ 中藥材清單

目前系統收錄以下類別的中藥材/方劑/保健品：

### 單味中藥材（25項）
- **活血類**：當歸、丹參、川芎、紅花、桃仁、益母草、銀杏
- **補氣類**：人參、西洋參、黃耆、甘草
- **清熱類**：黃芩、黃連
- **補益類**：何首烏、枸杞、五味子
- **溫陽類**：附子、乾薑、肉桂
- **其他**：葛根、麻黃、半夏、決明子

### 常見方劑（3項）
- 四物湯（補血）
- 十全大補湯（補益）
- 血府逐瘀湯（活血）

### 保健品（3項）
- 魚油（Omega-3）
- 紅麴
- 薑黃

> 💡 **找不到您使用的品項？**  
> 系統目前收錄常見高風險品項，若找不到您使用的中藥，建議直接諮詢醫師或藥師。

## 🏥 西藥分群說明

系統會將您的西藥處方自動分類為以下風險群組：

1. **RX_A：抗凝血/抗血小板**
   - Warfarin、Aspirin、Clopidogrel 等
   - 主要風險：出血

2. **RX_B：降血糖藥**
   - Insulin、Metformin、Sulfonylurea 類等
   - 主要風險：低血糖

3. **RX_C：降血壓藥**
   - ACEI、ARB、Beta-blocker 等
   - 主要風險：血壓、電解質

4. **RX_D：鎮靜安眠/中樞抑制**
   - Zolpidem、Benzodiazepines 等
   - 主要風險：嗜睡、跌倒

5. **RX_E：利尿劑**
   - Furosemide、Hydrochlorothiazide 等
   - 主要風險：電解質失衡

6. **RX_F：免疫抑制/抗癌藥**
   - Cyclosporine、Methotrexate 等
   - 主要風險：藥物代謝干擾

> ⚠️ **注意**：目前使用關鍵字匹配，可能有部分藥物未能正確分類。

## 🔍 常見問題 FAQ

### Q1: 我的中藥材不在清單中怎麼辦？
**A:** 系統目前收錄常見高風險品項。若找不到，建議：
- 就醫時直接告知醫師
- 聯繫藥師諮詢專線
- 未來版本會持續擴充清單

### Q2: 顯示「無法取得西藥資料」？
**A:** 可能原因：
- FHIR 伺服器無 MedicationDispense 資料
- 您最近沒有西藥處方記錄
- 網路連線問題

### Q3: 評估結果顯示低風險，就代表完全安全嗎？
**A:** 不完全是。低風險表示：
- 目前規則表中無已知交互作用
- 但不代表「完全無風險」
- 仍建議告知醫師所有用藥

### Q4: 可以分享評估結果給醫師嗎？
**A:** 可以。建議：
- 使用「匯出報告」功能（開發中）
- 或直接在就診時出示手機畫面
- 口頭告知即可，不一定需要書面

### Q5: 如果發現高風險該怎麼辦？
**A:** 請依建議行動：
1. **立即諮詢**：儘速（24-48小時內）聯繫醫師或藥師
2. **留意症狀**：注意系統提示的警訊症狀
3. **不要自行停藥**：等待專業意見
4. **紀錄症狀**：如有不適請記錄時間與狀況

### Q6: 系統會自動通知我嗎？
**A:** 目前版本：
- 需主動進入評估頁面
- 未來可能加入推播通知（僅中/高風險）
- 通知不含敏感資訊

### Q7: 評估要收費嗎？
**A:** 目前為研究/試用階段，不收費。
